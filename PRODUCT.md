# Product

## Register

product

## Users

People moving messages and files between their own devices or nearby devices
on the same local network. They need a direct, low-friction interface that works
well on desktop and mobile without requiring cloud storage.

## Product Purpose

Quick Send provides persistent room chat and direct LAN file transfer. The
service device owns identity, room, membership, and message data in SQLite,
while file bytes travel directly between browsers.

## Brand Personality

Direct, calm, and practical. The interface should feel familiar immediately
and keep attention on rooms, messages, and transfer actions.

## Anti-references

Avoid cloud-storage dashboards, decorative SaaS interfaces, nested cards,
standalone pages for short secondary tasks, and custom controls that replace
familiar platform behavior.

## Design Principles

- Keep the primary room and chat workflows visible.
- Present secondary tasks in context without replacing the current screen.
- Use one consistent HeroUI component vocabulary across desktop and mobile.
- Prefer direct labels, compact layouts, and clear state feedback.
- Make mobile interactions reachable and native-feeling.

## Accessibility & Inclusion

Preserve keyboard navigation, visible focus states, semantic labels, sufficient
contrast, touch-friendly targets, and reduced-motion behavior.
